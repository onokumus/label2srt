## Install
Install globally with [npm](https://www.npmjs.com/)

```sh
$ npm install --global label2srt
```

## Usage

```sh
label2srt <inputfile.txt> [<outputfile.srt>]
```
## Options
+ inputfile  : required
+ outputfile : optional
