# label2srt
>Converts Audacity labels (txt files) into SubRip (srt files)

## Installation
Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --global label2srt
```

## Usage

```sh
label2srt <inputfile.txt> [<outputfile.srt>]
```

## Options
+ inputfile  : required
+ outputfile : optional
